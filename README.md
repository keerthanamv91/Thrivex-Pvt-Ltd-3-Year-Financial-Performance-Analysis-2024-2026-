# Financial Performance Analysis & Dashboarding – ThriveX Pvt Ltd (2024–2026)
This project presents a comprehensive financial analysis of Thrivex Pvt Ltd over a three-year period (2024–2026) using Microsoft Excel. The dataset consists of detailed account statements, including transaction dates, customer details, transaction types, deposits, and withdrawals.

The goal of this project is to transform raw financial data into meaningful insights by applying data cleaning, analysis, and visualization techniques.

Key highlights of the project include:

Cleaning and structuring raw financial data for analysis
Analyzing yearly trends in deposits and withdrawals
Categorizing and evaluating transaction types (FT, TFR, CLG, etc.)
Identifying high-value transactions and recurring patterns
Building interactive dashboards for clear financial reporting

Through this project, I demonstrate hands-on experience in Advanced Excel, including data preprocessing, pivot tables, financial analysis, and dashboard creation. This analysis helps in understanding business performance and supports data-driven decision-making.

This project is part of my portfolio focused on data analytics and business intelligence.
